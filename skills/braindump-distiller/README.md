# braindump-distiller

Turns unstructured notes, requirements, or ideas into an actionable plan with a clear
goal, ordered small tasks, assumptions, risks, and questions to resolve. It returns
Markdown by default and can create an interactive HTML checklist when requested.

## When to use

Use when the user needs to organise a brain-dump, decide where to start, or break a
messy idea into a plan. It is a planning aid, not an implementation or architecture
review workflow.

## Output

**Summary**, a phased **Plan**, **Risks**, and a **Next action**. HTML plans use the
bundled template and retain browser-local checklist progress and notes.
